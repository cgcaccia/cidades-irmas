Bem-vindo!

Esta página foi criada para apresentar dados de uma pesquisa que tem como objetivo analisar a implementação de políticas públicas a partir dos acordos de irmanamento de cidades. Inicialmente, buscou-se coletar os dados junto ao município de Florianópolis. Em um segundo momento, a intenção é expandir a pesquisa para todas as capitais.
